Essa é uma página para exercitar códigos básicos de html, css e JavaScript, tais como tags de título, parágrafo, listas, imagens e etc.

Usando a tag 'strong'

A tag 'strong' destaca o texto em um tom mais escuro permitindo destacar
a parte do texto que merece ser vista com mais atenção.
